# Documentation

This folder contains the curated project documentation for the Cardputer ADV CarQuest firmware.

## Files

- `CARQUEST_DESIGN.md`: full CarQuest game setting, mechanics, progression, monster families, treasure, equipment, events, and long-term roadmap.
- `CARQUEST_DESIGN_ZH.md`: Chinese translation of the CarQuest design bible.
- `FIRMWARE_STATUS.md`: current implementation status against the design roadmap.

## Current Focus

The firmware has completed the prototype stabilization work and most content-expansion systems. The next major design/implementation area is Camp Depth:

- inventory detail page
- merchant events
- relic management
- quest log
- longer event review
